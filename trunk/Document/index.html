GLADE, Overview

Welcome to Annex E for GNAT

This Project is dedicated to provide an up to date Annex E interface for the current GNAT, which for some unknown reason is called GLADE and has nothing to to interface builder of the same name

How to get GLADE

GLADE works very closely with the compiler and a single GLADE compile is only guaranteed to work with the compiler it was created with.

Get a Compiler

First, if you have not compiled you own compiler yet, you should do so. The following script might help you:

function Checkout_From_CVS ()
    {
    local -r in_Dir=${1};
    local -r in_Branch=${2};

    pushd ${in_Dir};

        cvs -qz9 checkout -P ${in_Branch} gcc | tee ../checkout.${in_Branch}.log

    popd;

    return;
    }

function Update_From_CVS ()
    {
    local -r in_Dir=${1};

    mkdir ${in_Dir};
    pushd ${in_Dir};

        cvs -qz9 update -C -d | tee ../update.log

    popd;

    return;
    }

function Configure_And_Compile ()
    {
    local -r in_Dir=${1};
    local -r in_Config=${2};

    mkdir ${in_Dir}/${in_Config}; Heimatseitefür www.krischik.com
    pushd ${in_Dir}/${in_Config};

        ../gcc/configure                            \
            --prefix=/opt/ada                       \
            --enable-libada                         \
            --enable-shared                         \
            --enable-threads=gnat                   \
            --enable-languages=ada,c,f77,c++        \
            --target=${in_Config}

        make -j8 | tee ../compile.${in_Config}.log

    popd;

    return;
    }

declare -x CVS_RSH=ssh
declare -x CVSROOT=:ext:anoncvs@savannah.gnu.org:/cvsroot/gcc

Update_From_CVS gnu_3.5/gcc
Update_From_CVS gnu_3.4/gcc

Configure_And_Compile gnu_3.4 i686-pc-linux-gnu
Configure_And_Compile gnu_3.4 x86_64-suse-linux-gnu

Configure_And_Compile gnu_3.5 i686-pc-linux-gnu
Configure_And_Compile gnu_3.5 x86_64-suse-linux-gnu

The compiler will be installed to /opt/ada.

Get GLADE

Best is to download A.S.I.S from cvs Archive. See https://sourceforge.net/cvs/?group_id=105524 how to do that.

Compile GLADE

In order to create a working GLADE. you have to copy some of the files from them original compiler sources to the ./gnat directory within GLADE. You need to set GNATSRC to the gnat sources and then use make gnat to copy.

After that just say make install. The GLADE library will be installed to /opt/ada/install/asis and the GLADE tools to /opt/ada/bin.
